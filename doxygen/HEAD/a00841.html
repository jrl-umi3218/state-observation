Detailed Description

The base class for algebraic sensors.

Algebraic sensors are sensors which depend only on the state value and the current time and do not have internal dynamics (or a dynamics which converges fast enough to be ignored). This class implements mostly the containers and the interface to algebraic sensors. Algebraic sensors must be derived from this class.
